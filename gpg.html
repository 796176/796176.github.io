Gnu Privacy Guard

My key is available in armored and binary formats.

Importing/Updating key

  1. Retrieve the key from the server:

    curl -o 796176.asc https://796176.github.io/static/keys/796176.asc

  2. Add my key to your keyring:

    # the same command updates the previously added key

    gpg --import 796176.asc

  3. After validating the key fingerprint, sign it with your own key:

    gpg --lsign-key 796176

    Alternatively, in case you don't have your own key it's possible - although not recommended - to grant the key the highest trust level:

    gpg --edit-key 796176

    gpg> trust

    # this makes you trust my key and every key i sign https://www.gnupg.org/gph/en/manual.html#AEN335

    Your decision? 5

    gpg> save

Verying the message

There are two types of signatures -- text based and detached. For verifying the text based variant save the message, starting from and including the -----BEGIN PGP SIGNED MESSAGE----- line and ending with and including the -----END PGP SIGNATURE----- line. Then execute: gpg --verify message where message is the file containing the signed message. Text based signature example.

For verifying the detached signature, save the file containg it and execute: gpg --verify message where message is the file containing the detached signature. Detached signature example.
